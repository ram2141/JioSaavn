# JioSaavn
## _Listen Your favourites_
A Music App Developed using MERN stack and [JiosaavnAPI]



## Tech used
- Reactjs (Frontend)  
- Redux
- Expressjs (Backend)
- Used [JiosaavnAPI]

### `Features`
- Enjoy Ads free Music
- Trending List
- Search your favourite Song
- Add your songs to Library
- Add To play Next
- Enjoy Different Radio Stations
- Lyrics
- Choose Song Quality
